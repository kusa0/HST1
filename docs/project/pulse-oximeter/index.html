パルスオキシメーターは医療機関のみならず登山愛好者やスポーツ愛好者などにも広く普及しています。HSTはこうした一般人向けはもとより高度なケアを必要とする医療機関向けのパルスオキシメータの開発実績があります。

医療機関向け機器として薬機法の承認を受けるにはISO規格に則り臨床評価が必要となります。様々な人種や肌色の違う人間を対象に低酸素状態からの観血的な動脈血ガス検査との相関を取る必要があります。HSTはこの規格に適合していると海外の評価機関にて評価された製品を開発しました。
